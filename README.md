# BezierAnimation
first

贝塞尔曲线动画，基本能满足日常动画需求

**1.添加JitPack仓库 在当前项目等根目录下的 build.gradle 文件中添加如下内容:**

`allprojects {
    repositories {
        ...
        maven { url "https://jitpack.io" }
    }
}`

**2.添加项目依赖,复制q876625596/BezierAnimation在jitpack.io上查看最新版本**

dependencies {
        compile 'com.github.q876625596:BezierAnimation:1.0.0'
}
